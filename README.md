# Previsão do Tempo 🌤️

Um aplicativo web dinâmico para consulta de previsão do tempo, desenvolvido com React. O projeto permite aos usuários buscar e visualizar as condições climáticas atuais, bem como acompanhar a previsão estendida para os próximos 5 dias, com uma interface ilustrada por ícones climáticos customizados.

## 🚀 Funcionalidades

- Busca e exibição dos dados climáticos atuais (temperatura, clima).
- Previsão do tempo estendida para os próximos 5 dias.
- Ícones climáticos dinâmicos que mudam de acordo com o tempo (sol, chuva, neve, névoa, noite, nublado, trovoada).
- Integração com API externa de dados meteorológicos via Axios.

## 🛠️ Tecnologias Utilizadas

- **[React](https://react.dev/)** (v19.1.0)
- **[Axios](https://axios-http.com/)**
- **CSS3**
- **Create React App**

## ⚙️ Pré-requisitos

Para rodar este projeto localmente, você vai precisar do **[Node.js](https://nodejs.org/)** (que já inclui o gerenciador de pacotes npm) instalado na sua máquina.

## 📦 Instalação e Execução

1. Clone o repositório para a sua máquina:
   ```bash
   git clone https://github.com/gabrielaanascimento/previsaodotempo.git
   cd previsaodotempo
   npm install
   npm start
   ```
## 🧪 Scripts Disponíveis
No diretório raiz, você pode executar os seguintes comandos nativos do projeto:

npm start: Inicia o aplicativo em modo de desenvolvimento.

npm test: Executa os testes unitários configurados via Jest/Testing Library.

npm run build: Compila o aplicativo para produção na pasta build, otimizando os arquivos para melhor performance e minificação.

npm run eject: Remove a dependência de build única do Create React App, copiando todos os arquivos de configuração para o projeto (Ação irreversível).
